disciplinasOferecimento()

Posgraduacao.php : 174

Retorna a lista de disciplinas em oferecimento de uma determinada área de concentração.

public static disciplinasOferecimento( $codare : int [, $data_ini : string = null ] [, $data_fim : string = null ] ) : array

Se $data_ini não for informado, pega a data corrente. Se for informado pegará as disciplinas oferecidas no semestre que contém a data. O formato pode ser qualquer aceito por datetime(). Se $data_ini for informado, o comportamento de $data_ini é alterado. $data_ini e $data_fim passam a ser as datas de início e data de fim do período de busca da disciplina, no formato aceito pelo BD (YYYYMMDD).

Parameters
$codare : int

Código da áreada PG.

$data_ini : string = null

(opcional) Data na qual vai buscar os limites do semestre. Caso seja fornecido $data_fim então será a data inicial do intervalo.

$data_fim : string = null

(opcional) data final do intervalo de busca.
